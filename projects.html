Mihir Gadgil (मिहिर गाडगीळ)

Projects & Workshops

  • Analysis of Mixed Quote Representation in News Sources, Natural Language Processing Project, GWU, Fall 2019
    • Quantitative analysis of partial representation of full quotes.
    • Web scraping for data collection.
    • Analysis using Jaccard similarity and k-means clustering.
  • American Sign Language Recognition using CNN, GWU, Fall 2019
    • Recognize basic alphabet hand signs from images.
    • Use pre-trained and fine tuned ResNet model from PyTorch.
  • Bayesian Analysis of Play Store App Ratings, GWU, Fall 2019
    • Hierarchical modelling of app ratings based on installs, reviews and price.
    • Generalized Logistic Regression model created using Markov Chain Monte Carlo (MCMC) in JAGS.
  • Earthquake Time Prediction, Machine Learning Project, GWU, Spring 2019
    • Entry for the LANL Earthquake Prediction competition.
    • Trained models including linear regression, elastic net and neural networks to predict earthquake times using real time seismic data.
    • Model selection and hyperparameter tuning using random forest.
  • Traffic Simulation with Cellular Automata, HPC and Parallel Programming Project, GWU, Spring 2019
    • Two lane traffic simulation using cellular automata.
    • Implements the algorithm described in this paper.
    • Parallelised using OpenMP in C.
  • Olympians, Data Visualisation Project, GWU, Spring 2019
    • Data exploration tool for the summer Olympic games.
    • Uses d3 for visualisation.
  • Prediction of Kickstarter Campaign Success, Data Mining Project, GWU, Fall 2018
    • Analyzed a Kickstarter campaigns dataset.
    • Built a prediction model for success using logistic regression based on catergory of the project, country of origin, duration of campaign and number of backers.
  • Prediction of a country's performance in Olympics, Data Science Group Project, GWU, Fall 2018
    • Built a linear regression prediction model for proportion of medals won by a country based on economic factors like GDP, Population, host advantage as found in a previous project and performance in the previous Olympics.
  • Does the host country of Olympics have an advantage? Data Science Group Project, GWU, Fall 2018
    • Analyzed data from summer Olympics to determine whether countries win more than their average when they’re hosting Olympics using a t-test.
  • Interaction of Catalytic Active Motors with Surface, Final Year Project, IISER Bhopal, 2016 - 17
    • Worked on developing a numerical simulation in C.
    • Simulation consists of a system of 10,000 particles interacting with each other and a wall through Lennard-Jones potential.
    • Involved grid search for nearest neighbors, periodic boundary conditions and application of Anderson thermostat.
  • XX SERC Preparatory School in Theoretical High Energy Physics, IISER Bhopal, July 2015
  • Application of Bounded Bhattacharyya Distance for Gravitational Wave Detection, IAS Summer Research Fellowship, at IIT Gandhinagar, Summer 2014
    • Developed application of BBD to signals.
    • Found out how BBD differs from other distance measures like KL divergence.
    • Work done in this project is included in the publication mentioned above.
